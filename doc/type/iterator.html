Abstract iterator

The class iterator is an abstract iterator designed in the style of Java Iterator.

The iterator should be able to iterate over an collection using following idiom:

@code{.f90}

class(iterator), allocatable :: iter class(*), pointer :: d allocate(iter, source=collection%iterator())

do while (iter%has_next()) d => iter%next() operate on d
end do

@endcode
